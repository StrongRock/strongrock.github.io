利用Hexo与github打造个人博客

第一步 注册github

在github上注册一个账号，当然，如果已经有了github账号则可以跳过这一步。

第二步 新建一个repository

新的仓库的名字应为如下形式：

账号名.github.io

进入当前仓库的setting页面，找到GitHub Pages，会发现一行提示：

Your site is published at https://strongrock.github.io/

第三步 配置本地环境

安装node.js

很简单，只需要访问node.js官网下载相应的版本安装即可。
安装完成后执行

验证是否安装成功。

安装Git

同样，只需要访问Git官网下载相应的版本即可安装。

安装成功后，

  • 在windows下右键菜单中会出现Git GUI Here和Git Bash Here。

  • 在Linux下可直接在Terminal中输入git命令验证

安装Hexo

在合适的目录下建立一个文件夹，用于存放你的博客数据，比如 d:/blog。
进入此文件夹所在目录，执行

npm install hexo -g

开始安装hexo。稍等片刻，待执行完毕后，输入

hexo -v

检查hexo是否安装成功。

待确认安装成功后，执行

hexo init

初始化该文件夹。

执行hexo g，生成页面。

执行hexo s，开启服务器，正式体验Hexo。

Hexo的默认端口是4000，此时在浏览器中输入：

http://localhost:4000

我们的博客便展示在眼前了。

在github上设置SSH Key

登陆github，在setting页面中进入SSH and GPG keys栏目，添加新的SSH Key。即，将id_ras.pub文件中的内容添加到SSH Keys中即可。

在Git bash中执行：

git -T git@github.com

如果看到Hi,【你的用户名】，则表明SSH配置成功。
